How Switch Statements Work

A switch statement in JavaScript provides a way to execute different code blocks based on different conditions.

Syntax: The switch expression is evaluated once and compared with the values of each case. If there's a match, the associated block of code is executed.

Break keyword: The break statement terminates the switch statement. Without it, execution would "fall through" to the next case.

Default case: Though not used in this example, a default case can be added to execute if no cases match.
